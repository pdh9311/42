# get_next_line

> get_next_line 함수는 파일의 내용 한줄 읽는 함수를 만들어보는 과제입니다.

## 이해도가 향상되는 부분
- 메모리 누수에 대한 이해
- 파일 디스크럽터에 대한 이해

## 구현한 get_next_line을 테스트하는 방법
- 테스트 실행 파일을 생성하는 명령어 : `gcc get_next_line.c get_next_line_utils.c main.c`
- 테스트 실행 명령어 : `./a.out`

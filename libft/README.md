# LIBFT

> libft 는 C에서 사용되는 라이브러리를 직접 만들면서 포인터에 대한 이해와 메모리를 직접 할당에 대한 이해가 향상시킵니다. 또한 이렇게 만든 라이브러리를 하나의 정적 아카이브 파일 (*.a)로 만듬니다.

## 제한 조건
- `printf`를 사용하지 않고 저수준 함수인 `write`를 사용합니다.

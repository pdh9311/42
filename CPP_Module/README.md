# CPP_MODULE

> CPP_Module 은 C++ 에 대한 기본적인 문법을 익히고, 객체지향에서 볼 수 있는 클래스, 상속, 인터페이스, 접근지정자, ... 를 이해하고 주어진 문제를 해결해야합니다.

## 제한 조건
- `STL` 사용가능한 문제에서만 사용할 수 있습니다.
- `c++98` 버전으로 작성해야 합니다.
- `clang++` 컴파일러를 사용합니다.
